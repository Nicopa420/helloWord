# EjemploProyecto.n

> Un README limpio y elegante para mostrar cómo puede lucir un proyecto en GitHub.

---

## 🚀 Descripción

Este repositorio es un proyecto de demostración que contiene un ejemplo sencillo de `Node.js` y `TypeScript`, junto con un pequeño script `Python` llamado `helloWorld.py`.

La idea es presentar un README bien estructurado y visualmente agradable, ideal para un perfil profesional en GitHub.

---

## 🎯 Contenido del proyecto

- `package.json` — configuración de npm y dependencias.
- `tsconfig.json` — configuración de TypeScript.
- `app.ts` — punto de entrada TypeScript esperado para el proyecto.
- `helloWorld.py` — script Python de demostración.

---

## 🧩 Instalación

Si quieres ejecutar el proyecto con Node.js:

```bash
npm install
```

> Nota: este repositorio está configurado para `commonjs` y usa `typescript` como dependencia de desarrollo.

---

## ▶️ Uso

### Ejecutar el ejemplo de TypeScript

```bash
npx tsc
node ./dist/app.js
```

### Ejecutar el script Python

```bash
python helloWorld.py
```
